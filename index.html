  • 关于hexo丢失css样式问题

    2023-03-21 |
    • { hexo }

    在执行了hexo的部署命令后出现了css样式的问题，导致页面只有html结构

    解决问题

    快速排查

    在浏览器中按f12，打开开发者工具。
    可以看到获取css文件返回404

    这是因为我们的根目录不是 https://lvivinnie.github.io/
    而是https://lvivinnie.github.io/blog.github.io/
    所以正确的css样式表的路径应该是https://lvivinnie.github.io/blog.github.io/css....style.css
    通过我们访问页面的首页url就能够找到css对应路径

    如何解决

    将hexo项目中的_config文件中的url修改为

    生成部署

    最后注意

    git上托管网站在部署以后有一定延时，请耐心等待几分钟再刷新！
